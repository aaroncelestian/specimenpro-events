# SpecimenPro Events

This repository contains the events page content for the [SpecimenPro iOS app](https://aaroncelestian.github.io/SpecimenPro-Webpage/).

## About

SpecimenPro is an iOS application for mineral specimen collectors and enthusiasts. This repository hosts the events data and resources that are displayed within the app.

## Structure

- `specimens/` - Specimen data and information
- `images/` - Event and specimen images
