# Assets Directory

This directory contains media files for SpecimenPro events.

## Structure

- **images/** - Specimen photos and event images
- **audio/** - Audio notes and specimen descriptions

## Usage

When using the Event Manager application:
1. Click "Browse" next to Photo URL or Audio Note URL fields
2. Select a file from your computer
3. The file will be automatically copied to the appropriate folder
4. The URL will be set to the GitHub Pages location

## GitHub Pages URLs

Files in this directory are accessible via:
- Images: `https://aaroncelestian.github.io/specimenpro-events/assets/images/[filename]`
- Audio: `https://aaroncelestian.github.io/specimenpro-events/assets/audio/[filename]`

## Notes

- Supported image formats: JPG, PNG, GIF, BMP, WebP, SVG
- Supported audio formats: MP3, WAV, M4A, AAC, OGG
- Keep filenames simple (no spaces or special characters recommended)
