# üìù How to Create Events for SpecimenPro

This guide will help you create events that can be imported into the SpecimenPro app. **No programming knowledge required!**

## üöÄ Quick Start

### Step 1: Open the Event Creator
1. Find the file called `event_creator.html` on your computer
2. Double-click it to open it in your web browser (Chrome, Safari, Firefox, etc.)
3. You should see a colorful form titled "SpecimenPro Event Creator"

### Step 2: Fill Out Your Event

#### Event Information
Fill in the basic details about your event:
- **Event Title**: Give your event a catchy name (e.g., "Minerals of India")
- **Description**: Describe what the event is about
- **Event Type**: Choose from:
  - Exhibit (for museum displays)
  - Scavenger Hunt (for collecting activities)
  - Competition (for competitive events)
  - Workshop (for educational sessions)
- **Status**: Choose the current state:
  - Draft (still working on it)
  - Upcoming (scheduled for the future)
  - Active (happening now)
  - Ended (already finished)
- **Start Date & Time**: When does the event begin?
- **End Date & Time**: When does the event end?

#### Location Information
Tell us where the event takes place:

- **Location Name**: The name of the place (e.g., "Natural History Museum of Los Angeles County")
- **Latitude & Longitude**: The GPS coordinates
  - **How to find coordinates:**
    1. Go to Google Maps (maps.google.com)
    2. Search for your location
    3. Right-click on the location
    4. Click on the coordinates that appear (they look like: 34.0171, -118.2889)
    5. Copy the first number (latitude) and second number (longitude)
- **Search Radius**: How far from the location should people be able to see this event? (in meters)
  - 100 meters = about 1 city block
  - 500 meters = about 5 city blocks
  - 1000 meters = 1 kilometer
- **Always Visible**: Check this box if this event should ALWAYS show up, no matter where people are

### Step 3: Add Specimens (Optional)

Specimens are the items people can collect during your event.

1. Click the "**+ Add Specimen**" button
2. Fill in the specimen details:
   - **Specimen Name**: What is it called? (e.g., "Heulandite")
   - **Locality**: Where is it from? (e.g., "Poona, India")
   - **Description**: Brief description
   - **Rarity**: How rare is it? (Common, Uncommon, Rare, Legendary)
   - **Chemical Composition**: The chemical formula (if you know it)
   - **Fun Facts**: Interesting tidbits about the specimen
   - **Story**: Tell the full story of this specimen
   - **Photo URL**: Link to a photo (if you have one online)
   - **Audio Note URL**: Link to an audio file (if you have one)
3. Click "**Add Specimen**"
4. Repeat for each specimen you want to add

### Step 4: Add Badges (Optional)

Badges are achievements people can earn during your event.

1. Click the "**+ Add Badge**" button
2. Fill in the badge details:
   - **Badge Title**: Name of the achievement (e.g., "First Scan!")
   - **Description**: What did they accomplish?
   - **Icon Name**: The icon to use (e.g., "star.fill", "trophy.fill")
   - **Color**: Choose a color for the badge
   - **Requirement Number**: How many times do they need to do something?
   - **Requirement Type**: What do they need to do?
     - Collect Count (collect X specimens)
     - Visit Count (visit X times)
     - Time Spent (spend X minutes)
3. Click "**Add Badge**"
4. Repeat for each badge you want to add

### Step 5: Download Your Event

1. Scroll to the bottom of the page
2. Click the green "**üì• Download Event File**" button
3. A file will be downloaded to your computer (it will be named something like `your_event_name_event.json`)
4. **Save this file** - you'll need to send it to the event manager!

### Step 6: Send Your Event

1. Find the downloaded file (usually in your Downloads folder)
2. Send it via email or file sharing to the person managing the SpecimenPro events
3. They will import it into the system!

## üéØ Tips for Creating Great Events

### Writing Descriptions
- Be clear and engaging
- Explain what makes this event special
- Include any special instructions

### Choosing Locations
- Make sure the coordinates are accurate
- Set a reasonable radius (not too small, not too large)
- Test the location on Google Maps first

### Adding Specimens
- Include interesting stories and fun facts
- Add photos if possible (they make it more engaging)
- Be accurate with scientific information

### Creating Badges
- Make achievements meaningful but achievable
- Use descriptive titles
- Choose appropriate icons and colors

## ‚ùì Common Questions

**Q: Do I need to fill in every field?**
A: No! Only fields marked with a * (asterisk) are required. Everything else is optional.

**Q: What if I make a mistake?**
A: No problem! You can always create a new event file. Just fill out the form again and download a new file.

**Q: Can I edit an event after creating it?**
A: The event creator doesn't support editing. But the event manager can edit imported events using the Event Manager application.

**Q: What if I don't have photos or audio files?**
A: That's okay! Leave those fields blank. You can always add them later.

**Q: How do I find icon names?**
A: Common SF Symbol icon names include:
- star.fill
- trophy.fill
- checkmark.seal.fill
- medal.fill
- flag.fill
- sparkles
- heart.fill

**Q: What format should dates be in?**
A: Just use the date/time picker in the form - it will automatically format them correctly!

## üÜò Need Help?

If you run into any problems:
1. Make sure you've filled in all required fields (marked with *)
2. Check that your dates make sense (end date should be after start date)
3. Verify your coordinates are correct on Google Maps
4. Contact your event manager for assistance
